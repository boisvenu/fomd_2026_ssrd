Abstract Emailer & PDF Tool

Approval Emails sent when:

  • Column Q (Approval Status) = Approved
  • Column U (Email Status) is blank

Rejection Emails sent when:

  • Column Q (Approval Status) = Not Approved
  • Column U (Email Status) is blank

Optional: Add reviewer comments in Column R (Review Notes) — included in the rejection email automatically.

Generate Abstract PDFs:

  • Only processes rows where Column S (PDF Status) is blank
  • Saves PDF to the Abstract Submissions Drive folder
  • Writes the Drive link to Column T (PDF Drive Link)

Assign Poster Numbers:

  • Only processes rows where Column Q = Approved and Column V (Poster Number) is blank
  • Assigns sequential P-001, P-002, … numbers
  • Safe to run multiple times — will not overwrite existing numbers
