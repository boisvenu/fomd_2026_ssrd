Email Notifications ▼

Approval emails — sent to rows where Approval Status = Approved and Email Status (col U) is blank.

Rejection emails — sent to rows where Approval Status = Not Approved and Email Status is blank. Reviewer Notes (col R) are included automatically.

Posters ▼

Assign Poster Numbers — assigns sequential numbers to all approved abstracts that are missing one, filling any gaps. Will never overwrite an existing number.

Fix Missing Poster Numbers — poster numbers are also assigned automatically when you approve an abstract in the review panel above. Use this only as a fallback if an approved row is still missing a number.

Abstract List ▼

Creates a PDF listing all approved abstracts sorted by poster number, with columns for #, first author (Last, First), and abstract title. All approved abstracts must have a poster number first.

Name Badges (Avery 5395) ▼

Prints name badges for all approved abstracts. Students come first (sorted by poster number), then supervisors (sorted alphabetically) — duplicates removed.

Student: name · [Supervisor] Lab · Poster #
Supervisor: Dr. [Name] · Department · Institute

Poster Tags (Avery 5395) ▼

Prints poster tags for all approved abstracts with assigned poster numbers, sorted by poster number. Each tag shows the poster number and student / supervisor last names.

Formatted for Avery 5395 (2 columns × 4 rows, 8 per sheet). In the print dialog, select Actual Size / 100% — no scaling.
